##Brainteaser project that is based on popular Trivia Game.

You can play it on https://brain-teaser.herokuapp.com/

Or install it locally:

##Server
Install server dependencies:
```
cd server
npm install
```
Seed the database:
```
npm run seeds
```
Run express:
```
npm run server:dev
```

##Client
```
cd ../client
npm install
npm run serve
```

you can see it on http://localhost:8080/
